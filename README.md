# 🍽️ Cognifyz Data Analysis Internship Project

## 📖 Project Overview

This repository contains the work completed during the **Data Analysis Internship** at **Cognifyz Technologies**.

The primary objective of this project is to perform **Exploratory Data Analysis (EDA)** on a restaurant dataset to uncover meaningful insights related to restaurant distribution, cuisines, ratings, pricing, online delivery services, geographic locations, and restaurant chains.

Using Python and its data analysis ecosystem, the raw dataset was cleaned, explored, analyzed, and visualized through multiple professional notebooks. Each notebook focuses on a specific business problem and presents clear visualizations along with analytical insights.

This project demonstrates practical skills in data preprocessing, statistical analysis, visualization, and business intelligence using real-world restaurant data.

---

# 🎯 Project Objectives

The main objectives of this project are:

* Understand the restaurant dataset through Exploratory Data Analysis (EDA).
* Discover hidden business patterns from restaurant data.
* Analyze customer preferences using ratings and cuisines.
* Study restaurant distribution across different cities.
* Explore online food delivery trends.
* Identify restaurant chains and evaluate their popularity.
* Analyze geographical distribution of restaurants.
* Present insights using professional visualizations.

# 💡 Key Business Insights

The analysis produced several important findings:

* North Indian cuisine is the most frequently served cuisine.
* Restaurant businesses are concentrated in major metropolitan cities.
* Online delivery services are available only for a subset of restaurants.
* Customer ratings vary significantly across different cities.
* Popular restaurant chains have multiple outlets and receive higher customer engagement.
* Geographic analysis shows restaurant clusters in densely populated urban areas.
* Price range analysis indicates that mid-range restaurants dominate the dataset.
